## The problem
Caring for an aging parent means juggling medications, appointments and updates — scattered across texts and sticky notes, with everyone out of sync.

## The solution
CareCircle puts the whole family on one shared timeline, and lets anyone ask a question that's answered from the actual care notes.

## ✨ What makes it different
| | |
|---|---|
| 🗓️ **Shared care timeline** | Medications, appointments and updates the whole circle sees in real time. |
| 💬 **Ask anything (RAG)** | "What did the doctor say about the dosage?" → answered from 10K+ care notes via pgvector semantic search, **with the source**. |
| 🔔 **Reminders & roles** | Medication reminders and role-based access for family + physicians. |

## 🏗️ Tech
Next.js + NestJS monorepo · PostgreSQL + **pgvector** · Gemini · real-time chat (Socket.IO).
